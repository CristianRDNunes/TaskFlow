🚀 TaskFlow - Gestor de Tarefas Inteligente
O TaskFlow é uma aplicação web minimalista e eficiente para a gestão de produtividade diária. O seu diferencial é o fluxo de trabalho "Liste primeiro, priorize depois", que permite anotar tarefas rapidamente e definir a urgência apenas quando necessário.

✨ Funcionalidades
Adição Rápida: Registe tarefas instantaneamente apenas com o texto (pressione Enter).

Edição de Prioridade Inline: Defina níveis (Alta, Médio, Baixa) diretamente na lista através de seletores coloridos.

Ordenação Inteligente: Tarefas urgentes sobem automaticamente; tarefas concluídas descem para o fim da lista.

Filtros Avançados:

Filtre por Status (Tudo, Pendentes ou Concluídas).

Filtre por Nível de Prioridade (Alta, Média ou Baixa).

Persistência Local: Os dados são guardados no localStorage do navegador, garantindo que não perde nada ao fechar a página.

Design Responsivo: Interface moderna e limpa construída com Tailwind CSS.

🛠️ Tecnologias Utilizadas
React.js: Biblioteca principal para a interface.

Tailwind CSS: Estilização rápida e moderna.

Lucide Icons: Conjunto de ícones minimalistas (em formato SVG).

Local Storage API: Para armazenamento de dados sem necessidade de base de dados externa.

🚀 Como publicar no GitHub Pages (Sem instalar nada)
Criar o Repositório: Vá ao GitHub, crie um novo repositório público.

Subir o Ficheiro: Faça o upload do seu ficheiro index.html.

Ativar o Site:

Vá a Settings > Pages.

Em "Branch", selecione main e clique em Save.

Aceder: Em cerca de 1 minuto, o GitHub fornecerá um link oficial para usar a sua aplicação em qualquer dispositivo!
